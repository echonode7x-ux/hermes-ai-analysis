## 信号1：用AI浏览器自动化接企业数据采集/表单填写单子

### 机会描述
browser-use（94,093星）和nanobrowser（12,988星）今天仍在GitHub Trending前列。这两个工具让AI agent能像人一样操作浏览器——点击按钮、填写表单、抓取数据、下载文件。单人可以用这套工具接：
- 企业数据采集（竞品监控、价格抓取、招聘信息聚合）：500-3000元/单
- 批量表单提交（招聘投递、问卷填写、注册账号）：200-1000元/单
- 网站自动化测试：1000-5000元/单

真实案例：browser-use的GitHub Issues里有大量用户在问"如何用这个接商业项目"，说明需求真实存在。

### 技术栈
- 工具：browser-use（Python）+ Playwright + Claude API / DeepSeek API
- 成本：DeepSeek API约0.5元/万token，一次自动化任务约0.1-0.5元；服务器0元（本地运行）
- 替代方案：nanobrowser（Chrome扩展，零代码，适合非程序员）；vercel-labs/agent-browser（Rust，性能更好）

### 执行路径（7天MVP）
- Day 1：安装browser-use，跑通官方demo（pip install browser-use，配置API key）
- Day 2：用browser-use实现一个"自动抓取淘宝商品价格"的脚本，录屏作为案例
- Day 3：在闲鱼/淘宝发布"AI数据采集服务"，定价99元/次，附案例视频
- Day 4：用nanobrowser做第二个案例——"自动批量填写Google Form"，发小红书
- Day 5：在V2EX/即刻发帖"我用AI帮你自动抓取任何网站数据"，接第一单
- Day 6：根据客户需求优化脚本，建立模板库（电商/招聘/房产各一套）
- Day 7：写一篇"我如何用AI浏览器自动化月入5000"的教程，引流

### 收益模型
- 悲观：1000元/月（每周接2-3单小单，99元/单）
- 中性：5000元/月（每周接5单，均价200元，含数据采集+表单填写）
- 乐观：15000元/月（接企业长期数据监控合同，3000元/月/客户，5个客户）

### 风险提示
- 反爬机制强的网站（如LinkedIn、部分电商平台）可能封IP，需要搭配代理IP（额外成本50-200元/月）
- 客户需求千差万别，前期需要大量定制化，时间成本高
- 纯数据采集容易被替代，需要往"数据分析+采集"方向升级

## 信号2：用MCP协议做广告投放自动化服务

### 机会描述
markifact-mcp（56星，5月3日创建）是一个MCP server，支持通过Claude/ChatGPT/Cursor直接操作Google Ads、Meta Ads、TikTok Ads、LinkedIn Ads，覆盖300+广告操作。这意味着：
- 不需要懂广告平台后台，用自然语言就能管理广告
- 可以帮中小商家做AI广告投放代运营：2000-8000元/月/客户
- 可以做"AI广告优化师"培训课程：99-299元/人

真实数据：markifact-mcp的topics包含google-ads-mcp、meta-ads-mcp、tiktok-ads-mcp，说明已经覆盖主流广告平台。

### 技术栈
- 工具：markifact-mcp + Claude Desktop / Cursor + 各广告平台API
- 成本：Claude Pro订阅20美元/月（约145元）；广告平台账号免费注册
- 替代方案：直接用各平台官方API + Python脚本（更灵活但开发成本高）

### 执行路径（7天MVP）
- Day 1：克隆markifact-mcp仓库，按README配置MCP server到Claude Desktop
- Day 2：用自己的Google Ads账号测试，用自然语言创建一个广告campaign
- Day 3：截图+录屏整个过程，做一份"AI帮你投广告"的案例展示
- Day 4：在小红书/抖音发"我用AI帮商家省了50%广告费"的内容，引流
- Day 5：在本地商家群/电商群发"免费帮你看广告数据"，获取第一批客户
- Day 6：用markifact-mcp帮1-2个商家做广告优化，收集before/after数据
- Day 7：用真实数据做案例，发布"AI广告代运营"服务，定价2000元/月

### 收益模型
- 悲观：2000元/月（1个客户，2000元/月代运营费）
- 中性：8000元/月（3个客户，均价3000元/月，含基础优化+周报）
- 乐观：25000元/月（8个客户，均价3000元/月 + 卖课程500元/人×10人）

### 风险提示
- 广告投放涉及真金白银，客户亏损可能引发纠纷，需要签合同明确责任
- 各广告平台API有配额限制，大量操作可能需要企业账号
- 需要一定的广告投放基础知识，纯靠AI不够，需要学习1-2周广告理论

## 信号3：用html-anything做AI内容设计服务

### 机会描述

这意味着你可以：
- 帮客户做AI生成的营销海报/小红书图：50-200元/张
- 帮客户做AI生成的数据报告/PPT：200-1000元/份
- 做"AI设计工具"的教程/模板包：9.9-99元/份
