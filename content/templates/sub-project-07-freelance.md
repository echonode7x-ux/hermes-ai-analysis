## Step 3：获客与转化（持续）

### 3.1 客户咨询模板
```
当有人咨询时，用以下话术：

开场：
"你好！感谢关注我的AI短剧～
我可以为你定制民间传说/仙侠风格的AI短剧视频。"

了解需求：
"请问你想做什么主题的呢？
比如：狐仙、龙女、白蛇传、聊斋、山海经等都可以。
视频时长希望多少秒？有没有参考的风格？"

报价：
"基础版¥99（30秒）/ 标准版¥199（1分钟）/ 定制版¥499（3分钟）
包含：AI画面+配音+字幕+配乐+修改
制作周期：3-10天（根据版本）"

成交：
"确定的话，你可以在闲鱼拍下这个链接：[链接]
拍下后我会第一时间开始制作～"
```

### 3.2 需求收集问卷
```
客户确认合作后，发送以下问卷：

1. 故事主题：（如：狐仙报恩）
2. 故事大纲：（简单描述，50字以内）
3. 视频时长：（30秒/1分钟/3分钟）
4. 风格偏好：（古风/动漫/写实/水墨）
5. 配音偏好：（男声/女声/自己录）
6. 用途：（自媒体/个人/商业）
7. 参考视频链接：（如有）
8. 特殊要求：（如有）
```

### 3.3 止损决策（Day 15）
```
第15天检查：

情况A：有成交
→ 继续，优化流程，提高效率

情况B：有咨询但无成交
→ 分析原因：
  - 价格太高？→ 降低到¥79试
  - 不信任？→ 增加作品展示
  - 需求不匹配？→ 调整服务内容

情况C：零咨询
→ 分析原因：
  - 流量不够？→ 增加发布频率
  - 标题不吸引？→ 优化标题
  - 平台不对？→ 尝试其他平台
→ 如果持续0咨询，考虑换赛道

保存记录：AI短剧/数据/Day15复盘.md
```
