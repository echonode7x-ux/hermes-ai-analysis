# 子项目4：第一条短剧制作（AI视频生成）

> 预计耗时：8小时（分2天） | 难度：⭐⭐⭐ | 成功率：75%
> 目标：完成12-15条AI视频素材，为剪辑做准备

---

## Step 1：准备分镜脚本（1小时）

### 1.1 细化剧本为分镜
```
打开子项目2中DeepSeek生成的剧本，逐画面细化：

每个画面需要明确：
├── 画面编号（1-8）
├── 时长（5秒/10秒）
├── 英文提示词（直接可用于可灵AI）
├── 中文旁白（定稿版）
├── 生成方式（文生视频 / 图生视频）
├── 备注（需要注意的细节）
```

### 1.2 分镜脚本模板
```
| # | 时长 | 生成方式 | 英文提示词 | 中文旁白 | 备注 |
|---|------|----------|------------|----------|------|
| 1 | 5s   | 文生     | [提示词]   | [旁白]   | 开场 |
| 2 | 5s   | 图生     | [提示词]   | [旁白]   | 角色首次出现用图生 |
| 3 | 10s  | 文生     | [提示词]   | [旁白]   | |
| ... |     |          |            |          |      |
```

### 1.3 关键角色参考图生成
```
对于有角色的镜头（尤其是主角），先用"文生图"生成参考图：

在可灵AI中选择"文生图"模式
输入角色描述提示词（不含动作，只描述外观）
生成5张，选最好的1张作为参考
保存到：AI短剧/AI生成素材/角色参考图/

重要：后续所有该角色的镜头都用这张图作为图生视频的输入
```

**✅ 验收标准：** 分镜脚本完成（8个画面），至少1张角色参考图

---

## Step 2：批量生成视频素材（6小时，分2天）

### 2.1 Day 1：生成画面1-4（3小时）
```
按分镜脚本顺序，逐个生成：

画面1：
├── 在可灵AI选择"文生视频"或"图生视频"
├── 粘贴英文提示词
├── 选择参数：5秒 / 1080p / 镜头运动：[根据分镜选择]
├── 点击生成 → 等待2-5分钟
├── 生成3条（点击生成3次）
├── 预览3条，选择最好的1条
├── 下载保存到：AI短剧/AI生成素材/画面1/
└── 记录：哪条最好，为什么

画面2-4：重复以上流程

关键提示：
- 如果某个画面生成效果都不好，调整提示词后重试
- 每个画面最多重试3次，3次都不行就换个画面描述方式
- 保存所有生成结果（包括不好的），供后续参考
```

### 2.3 Day 2：生成画面5-8（3小时）
```
同Day 1流程

额外注意事项：
- 画面5-8可能涉及动作场景，AI生成难度更高
- 动作场景建议用"图生视频"（先生成静态图再动起来）
- 如果动作场景实在不行，改用"慢动作"或"静止+特效"
```

### 2.4 生成失败的备选方案
```
如果某个画面反复生成失败：

方案A：简化画面描述（去掉复杂元素）
方案B：拆分成2个简单画面（5秒+5秒替代10秒）
方案C：用图片+Ken Burns效果替代视频（剪映中实现）
方案D：用纯色背景+文字旁白替代（最后手段）
```

**✅ 验收标准：** 至少10条可用素材（8个画面×至少1条+备用）

## Step 3：素材筛选与整理（1小时）

### 3.1 筛选标准
```
对每条生成的视频评估（1-5分）：

| 画面# | 候选1 | 候选2 | 候选3 | 最终选择 | 选择原因 |
|--------|-------|-------|-------|----------|----------|
| 1      |       |       |       |          |          |
| 2      |       |       |       |          |          |
| ...    |       |       |       |          |          |

评分维度：
- 画质清晰度
- 动作自然度
- 与剧本匹配度
- 风格一致性
```

### 3.2 整理素材
```
最终文件夹结构：
AI短剧/AI生成素材/
├── 最终选用/
│   ├── 画面1.mp4
│   ├── 画面2.mp4
│   └── ...
├── 备用/
│   ├── 画面1-候选2.mp4
│   └── ...
├── 角色参考图/
│   └── 主角参考.png
└── 失败记录/
    └── 失败原因分析.md
```

**✅ 验收标准：** 最终选用文件夹中有≥10条视频，按画面编号命名
