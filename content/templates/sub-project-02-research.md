# 子项目2：赛道调研与选题确定

> 预计耗时：3小时 | 难度：⭐⭐ | 成功率：90%
> 目标：确定首个故事选题，收集10个竞品参考

---

## Step 1：竞品调研（1.5小时）

### 1.1 抖音调研
```
打开抖音APP → 搜索以下关键词，每个记录Top5：

搜索词1："AI短剧"
├── 记录：账号名、粉丝数、最高播放量视频、内容风格
└── 截图保存

搜索词2："AI动画 民间传说"
├── 记录：同上
└── 截图保存

搜索词3："民间故事 短剧"
├── 记录：同上
└── 截图保存

搜索词4："聊斋 短剧"
├── 记录：同上
└── 截图保存
```

### 1.2 填写竞品分析表

打开DeepSeek，输入：
```
帮我整理一个竞品分析表格，包含以下列：
账号名 | 平台 | 粉丝数 | 最高播放量 | 内容风格 | 更新频率 | 变现方式

请留空让我填写。
```

然后手动填写你调研到的数据。

### 1.3 分析结论
```
回答以下问题（写在笔记本上）：
1. 哪种风格的民间传说播放量最高？（古风/搞笑/恐怖/感人）
2. 视频时长集中在多少秒？（30秒/60秒/90秒/3分钟+）
3. 头部账号的共同特点是什么？（配音/字幕/封面/标题）
4. 有没有AI做的？怎么判断的？
```

**✅ 验收标准：** 完成竞品分析表，明确哪种风格最受欢迎

---

## Step 2：选定赛道与风格（30分钟）

### 2.1 赛道确认
```
基于竞品分析，在以下子赛道中选1个：

A. 古风仙侠（狐仙/龙女/仙女）— 情感向，女性观众多
B. 志怪奇谈（聊斋/山海经）— 悬疑向，男女均衡
C. 民间智慧（阿凡提/包公）— 搞笑/推理，男性观众多
D. 神话传说（盘古/女娲/后羿）— 史诗向，全年龄

推荐首选：A或B（AI画面适配度最高）
```

### 2.2 视觉风格确认
```
在可灵AI中测试以下3种风格，选效果最好的：

风格1：写实古风
提示词：Chinese ancient painting style, ink wash, traditional Chinese aesthetic

风格2：动漫仙侠
提示词：Chinese anime style, xianxia fantasy, vibrant colors, Studio Ghibbi inspired

风格3：3D奇幻
提示词：3D render, Chinese fantasy, Pixar style, detailed textures

每个风格生成2条测试视频，对比选择
```

**✅ 验收标准：** 确定1个子赛道 + 1种视觉风格

---

## Step 3：选定首个故事（30分钟）

### 3.1 故事候选清单
```
基于选定赛道，列出5个候选故事：

仙侠赛道推荐：
1. 狐仙报恩 — 狐狸修炼成仙报答书生恩情
2. 龙女牧羊 — 龙王女儿被贬人间放羊
3. 田螺姑娘 — 田螺精暗中帮助贫穷农夫
4. 白蛇传（简化版）— 蛇妖与书生的爱情
5. 聊斋·画皮 — 妖鬼披画皮伪装美女

志怪赛道推荐：
1. 山海经·九尾狐 — 九尾狐的传说
2. 聊斋·聂小倩 — 书生与女鬼
3. 山海经·精卫填海 — 小鸟衔石填海
4. 干将莫邪 — 铸剑师的悲剧
5. 柳毅传书 — 书生为龙女传信
```

### 3.2 选题评估
```
对每个候选故事打分（1-5分）：

评估维度：
A. AI适配度 — 这个故事的画面好不好用AI生成？（非人类角色多不多）
B. 冲突强度 — 有没有强烈的戏剧冲突？（观众能不能看完）
C. 时长匹配 — 能不能在60秒内讲完？（太复杂的故事不适合短剧）
D. 差异化 — 有没有人做过类似的？（越少越好）

选总分最高的那个
```

### 3.3 用DeepSeek生成剧本
```
打开DeepSeek，输入以下提示词（根据你的选择替换[内容]）：

你是一个60秒短剧编剧。请写一个关于"[你选的故事名]"的短剧剧本。

要求：
1. 结构：开头悬念(5秒) → 故事展开(35秒) → 高潮(15秒) → 反转/钩子(5秒)
2. 拆分成6-8个画面
3. 每个画面包含：
   - 画面编号和时长
   - 视觉描述（用英文，详细描述画面内容，适合AI视频生成）
   - 中文旁白（每句不超过15字，有节奏感）
4. 角色：以非人类主角为主（狐仙/龙女/妖怪等）
5. 避免：复杂多人场景、手部特写、需要文字的画面
6. 情绪曲线：好奇 → 紧张 → 震撼 → 感悟

输出格式示例：
[画面1] 时长：5秒
视觉描述：A nine-tailed fox sitting on a moonlit rock...
旁白：月圆之夜，山中传来狐鸣。
```

**✅ 验收标准：** 获得一个完整的6-8画面剧本，旁白总时长约60秒

---

## Step 4：收集参考素材（30分钟）

### 4.1 收集10个参考作品
```
在抖音/快手中搜索你选定的故事名，收集10个相关视频：

记录每个视频的：
1. 视频链接
2. 画面风格（截图保存）
3. 配音风格（男声/女声/语速）
4. 字幕样式（字体/颜色/位置）
5. BGM类型（古风/紧张/温馨）
6. 评论区观众反馈（喜欢什么/吐槽什么）
```

### 4.2 建立风格参考库
```
在电脑上创建文件夹：
AI短剧/
├── 参考素材/
│   ├── 竞品截图/
│   ├── 配音参考/
│   ├── BGM参考/
│   └── 字幕参考/
├── 提示词库/
├── 剧本/
├── AI生成素材/
└── 成品/
```

**✅ 验收标准：** 文件夹结构创建完毕，10个参考作品已收集
