# 子项目6：发布上线与数据监测

> 预计耗时：3小时 | 难度：⭐⭐ | 成功率：90%
> 目标：视频在3个平台发布，24h后获取数据报告

---

## Step 1：准备发布素材（30分钟）

### 1.1 封面图制作
```
在剪映中截取视频最精彩的1帧作为封面：
1. 播放视频，找到画面最震撼/最好看的1帧
2. 点击"导出封面"（或截图）
3. 在剪映中新建封面：
   - 比例：9:16
   - 添加大字标题（3-5个字，如"狐仙报恩""月下奇缘"）
   - 字体：粗体，白色+红色/黄色描边
   - 位置：画面中央偏上
4. 导出封面图（1080×1920）
```

### 1.2 标题文案准备
```
标题公式：[悬念/冲突] + [故事名] + [钩子]

示例：
✅ "书生救了一只狐狸，当晚狐狸竟然..."
✅ "月圆之夜山中传来狐鸣，村民不敢出门"
✅ "她等了他一千年，终于等到..."
✅ "山海经中最痴情的妖，看完泪崩"

避免：
❌ "AI短剧《狐仙报恩》"（太平淡）
❌ "用AI做了一个短剧"（自嗨型）
```

### 1.3 标签准备
```
每个平台发布时添加标签：

通用标签：#民间传说 #中国故事 #短剧
内容标签：#[故事名] #狐仙 #仙侠
流量标签：#感动 #催泪 #反转

标签数量：5-8个（不要太多）
```

**✅ 验收标准：** 封面图+3个标题+标签列表准备完毕

---

## Step 2：发布到3个平台（1小时）

### 2.1 抖音发布
```
打开抖音APP → 点击"+" → 选择视频
设置：
├── 标题：选择准备好的标题
├── 封面：上传制作好的封面图
├── 标签：添加5-8个标签
├── 描述：补充1-2句话介绍故事背景
├── AI标注：如果有"AI生成"选项，勾选
└── 发布时间：晚7-9点（流量高峰）

点击发布
```

### 2.2 快手发布
```
打开快手APP → 点击"+" → 选择视频
设置：同抖音
发布
```

### 2.3 视频号发布
```
微信 → 发现 → 视频号 → 发表
设置：同抖音
发布
```

**✅ 验收标准：** 3个平台全部发布成功

---

## Step 3：24小时数据监测（1.5小时）

### 3.1 数据采集
```
发布24小时后，记录以下数据：

| 指标 | 抖音 | 快手 | 视频号 |
|------|------|------|--------|
| 播放量 | | | |
| 点赞数 | | | |
| 评论数 | | | |
| 转发数 | | | |
| 收藏数 | | | |
| 新增粉丝 | | | |
| 完播率 | | | |
| 平均播放时长 | | | |
```

### 3.2 数据分析
```
回答以下问题：

1. 哪个平台播放量最高？为什么？
2. 完播率是否>30%？
   - 是 → 内容吸引力合格
   - 否 → 开头3秒不够吸引人，需要优化
3. 评论区观众说什么？
   - 正面评论 → 记录他们喜欢什么
   - 负面评论 → 记录他们吐槽什么
   - 无评论 → 内容缺乏互动点
4. 哪个画面观众流失最多？
   - 在抖音创作者后台查看"观众流失点"
   - 记录：第几秒流失最多
```

### 3.3 数据报告
```
输出一份简单的数据报告：

# 首条视频数据报告

发布时间：YYYY-MM-DD HH:MM
平台数据：[表格]
分析结论：[3-5条]
改进方向：[下一条视频需要优化什么]

保存到：AI短剧/数据/首条报告.md
```
