# 子项目1：工具搭建与账号注册

> 预计耗时：3小时 | 难度：⭐ | 成功率：95%
> 目标：所有工具注册完毕，能生成第一条测试视频

---

## 前置检查

开始前确认你有：
- [ ] 智能手机（安卓/iOS均可）
- [ ] 电脑（Windows/Mac均可，能上网就行）
- [ ] 微信账号（用于登录剪映）
- [ ] 手机号（用于注册可灵AI）
- [ ] 支付方式（支付宝/微信，用于购买可灵专业版¥66/月）

---

## Step 1：注册可灵AI（45分钟）

### 1.1 打开网站
```
浏览器输入：klingai.com
点击右上角"登录/注册"
```

### 1.2 注册账号
```
方式A（推荐）：手机号注册
  → 输入手机号 → 获取验证码 → 输入验证码 → 完成

方式B：微信扫码
  → 用微信扫描页面二维码 → 手机端确认 → 完成
```

### 1.3 完成新手任务（获取免费积分）
```
注册后会弹出新手引导，完成以下任务：
1. 完善个人资料（+10积分）
2. 观看教程视频（+10积分）
3. 生成第一条视频（+20积分）
总共约获得40-60积分（可生成约10条5秒视频）
```

### 1.4 购买专业版
```
点击右上角头像 → "会员中心"
选择"专业版" → ¥66/月 → 支付宝/微信支付
购买后每日约66积分（约30条5秒视频）
```

### 1.5 测试生成
```
点击"文生视频"
输入提示词（复制粘贴）：
A mystical fox spirit with nine tails sitting on a rock in a moonlit bamboo forest, 
ethereal blue glow, gentle wind blowing through bamboo, 
cinematic wide shot, Chinese fantasy style

点击"生成" → 等待2-5分钟 → 查看结果
```

**✅ 验收标准：** 成功生成1条5秒视频，画面中有一只狐狸在竹林中

## Step 2：注册即梦AI（30分钟）

### 2.1 打开网站
```
浏览器输入：jimeng.jianying.com
用抖音账号登录（如果没有抖音，先注册一个）
```

### 2.2 了解免费额度
```
新用户有免费生成次数（具体数量以页面显示为准）
注意：高峰期可能需要排队数小时
建议：晚上提交任务，第二天早上取结果
```

### 2.3 测试生成
```
点击"视频生成"
输入与可灵相同的提示词
生成并对比效果
```

**✅ 验收标准：** 成功登录，了解免费额度数量，生成1条测试视频

## Step 3：下载安装剪映（30分钟）

### 3.1 电脑版（推荐主力工具）
```
浏览器输入：www.capcut.cn
点击"免费下载"
选择对应系统版本（Windows/Mac）
下载安装 → 用抖音账号登录
```

### 3.2 手机版（备用）
```
应用商店搜索"剪映"
下载安装 → 用抖音账号登录
```

### 3.3 熟悉界面（20分钟）
```
打开剪映，按顺序认识以下区域：

左侧：素材库
├── 视频素材（免费使用）
├── 音频素材（BGM、音效）
├── 文字模板
└── 特效/滤镜

中间：预览窗口
├── 播放/暂停按钮
└── 全屏预览按钮

底部：时间轴（最重要的区域）
├── 视频轨道（放视频/图片）
├── 音频轨道（放BGM/配音）
├── 字幕轨道（放字幕）
└── 右键菜单（分割、删除、复制）
```

### 3.4 测试基础操作
```
1. 随便导入一段视频到时间轴
2. 尝试"分割"功能（快捷键Ctrl+B / Cmd+B）
3. 尝试添加一个文字
4. 尝试导出（右上角"导出"按钮）
5. 导出参数：1080×1920 / 30fps / MP4
```

**✅ 验收标准：** 能独立完成导入→剪切→加文字→导出的完整流程

## Step 4：注册DeepSeek（15分钟）

### 4.1 打开网站
```
浏览器输入：chat.deepseek.com
手机号注册 → 验证码 → 完成
```

### 4.2 测试剧本生成
```
输入以下提示词（直接复制粘贴）：

你是一个60秒短剧编剧。请写一个关于"狐仙报恩"的短剧剧本。

要求：
1. 结构：开头悬念(5秒) → 故事展开(35秒) → 高潮(15秒) → 反转/钩子(5秒)
2. 拆分成6-8个画面
3. 每个画面包含：视觉描述（英文）+ 中文旁白（每句≤15字）+ 时长
4. 角色是狐仙（非人类主角）

点击发送 → 查看输出
```

**✅ 验收标准：** 获得一个格式完整的6-8画面剧本

## Step 5：注册发布平台账号（30分钟）

### 5.1 抖音
```
下载抖音APP → 手机号注册 → 完善资料
昵称建议格式："AI[赛道]短剧"（如"AI狐仙短剧""AI传说剧场"）
简介：用AI技术重新演绎中国民间传说 | 定制合作私信
```

### 5.2 快手
```
下载快手APP → 手机号注册 → 完善资料
昵称和简介与抖音保持一致
```

### 5.3 视频号（微信内）
```
微信 → 发现 → 视频号 → 创建视频号
昵称和简介与其他平台一致
```

**✅ 验收标准：** 3个平台全部注册完毕，昵称简介统一
