# 子项目5：剪辑合成与成片输出

> 预计耗时：4小时 | 难度：⭐⭐⭐ | 成功率：85%
> 目标：完成1条60秒成片，可直接发布

---

## Step 1：导入素材（30分钟）

### 1.1 打开剪映，新建项目
```
打开剪映 → 新建项目
项目名：[故事名]_v1
比例：9:16竖屏（手机全屏）
分辨率：1080×1920
```

### 1.2 导入所有素材
```
点击"导入"按钮
选择：AI短剧/AI生成素材/最终选用/ 下的所有视频
同时导入：角色参考图（备用）
导入完成后，素材出现在左侧素材库
```

**✅ 验收标准：** 所有素材导入完毕，在素材库可见

---

## Step 2：时间轴排列（1小时）

### 2.1 按分镜脚本排列
```
将素材从素材库拖到底部时间轴，按分镜脚本顺序排列：

时间轴示意：
[画面1:5s][画面2:5s][画面3:10s][画面4:5s][画面5:10s][画面6:5s][画面7:10s][画面8:5s]
|_____________ 总时长约55-60秒 _____________|
```

### 2.2 调整时长
```
如果某个画面太长/太短：
- 选中该片段 → 拖动边缘裁剪
- 或：选中 → 右键 → "变速" → 调整速度

目标：总时长控制在55-65秒
```

### 2.3 添加转场
```
在两个画面之间添加转场效果：
1. 点击两个片段之间的空隙
2. 选择"转场"效果
3. 推荐转场：
   - 古风场景：淡入淡出（柔和）
   - 紧张场景：闪白（冲击感）
   - 日常场景：无转场（直接切换）

注意：不要用太花哨的转场，会显得廉价
```

**✅ 验收标准：** 8个画面按顺序排列，总时长55-65秒

---

## Step 3：添加配音（1小时）

### 3.1 AI配音方式（推荐新手）
```
方法1：剪映内置AI配音
├── 点击"文字" → "新建文本"
├── 输入旁白文案
├── 选中文本 → 点击"文本朗读"
├── 选择声音：
│   ├── 古风传说：男声-沉稳/女声-温柔
│   ├── 悬疑志怪：男声-低沉
│   └── 语速：0.9x（稍慢更有韵味）
├── 生成后，音频自动出现在音频轨道
└── 调整音频位置，与对应画面对齐

方法2：自己录制（更有个人特色）
├── 找一个安静的地方
├── 用手机录音APP录制旁白
├── 语速：适中偏慢
├── 情绪：跟随故事起伏
├── 导入剪映 → 放到音频轨道
└── 与画面对齐
```

### 3.2 音画对齐
```
关键：旁白必须和画面对应！

操作：
1. 播放视频，找到画面切换点
2. 拖动音频片段，让旁白的开始时间和画面开始时间对齐
3. 如果旁白太长：加速音频（选中→变速→1.1x-1.2x）
4. 如果旁白太短：在旁白之间加停顿（用静音片段填充）

验收：播放一遍，旁白和画面完全同步
```

**✅ 验收标准：** 旁白与画面对齐，无错位

---

## Step 4：添加字幕（30分钟）

### 4.1 自动字幕
```
点击"文字" → "智能字幕" → "识别字幕"
选择语言：中文
剪映会自动识别配音内容生成字幕
```

### 4.2 校对字幕
```
逐条检查自动识别的字幕：
- 错别字修正
- 断句调整
- 时间轴对齐（拖动字幕条调整）

注意：AI自动识别准确率约90%，必须人工校对
```

### 4.3 字幕样式
```
推荐样式：
字体：思源黑体/思源宋体（免费商用）
大小：中等（不要太大会遮挡画面）
颜色：白色 + 黑色描边（保证在任何背景上都清晰）
位置：画面下方1/4处
```

**✅ 验收标准：** 字幕已校对，样式统一，无错别字

---

## Step 5：添加BGM和音效（30分钟）

### 5.1 选择BGM
```
在剪映中搜索免费BGM：

古风场景推荐关键词："古风""中国风""仙侠""悠扬"
悬疑场景推荐关键词："悬疑""紧张""恐怖""暗黑"
感人场景推荐关键词："感人""温暖""催泪""钢琴"

操作：
1. 点击"音频" → "音乐"
2. 搜索关键词
3. 试听 → 选择合适的
4. 拖到音频轨道（在配音下方）
5. 调整音量：BGM音量设为配音的20-30%（不要盖过配音）
```

### 5.2 添加音效
```
在关键画面添加音效增强氛围：

可选音效：
- 环境音：风声、水声、鸟鸣（搜索"环境音"）
- 动作音效：脚步声、开门声（搜索对应关键词）
- 情绪音效：心跳声、惊叹声（搜索"情绪音效"）

操作：
1. 点击"音频" → "音效"
2. 搜索并试听
3. 拖到对应画面的时间点
4. 调整音量（音效音量设为配音的40-50%）
```

**✅ 验收标准：** BGM全程有，关键画面有音效，BGM不盖过配音

---

## Step 6：调色与导出（30分钟）

### 6.1 统一调色
```
不同AI生成的画面可能色调不一致，用滤镜统一：

操作：
1. 选中一个片段
2. 点击"滤镜"
3. 选择一个滤镜（推荐古风用暖色调，悬疑用冷色调）
4. 调整滤镜强度（50-70%即可，不要太重）
5. 复制该片段的滤镜设置
6. 粘贴到所有其他片段

效果：所有画面色调统一，看起来像同一部作品
```

### 6.2 最终检查
```
播放一遍完整视频，检查：
- [ ] 画面和配音是否同步
- [ ] 字幕是否有错
- [ ] BGM音量是否合适
- [ ] 转场是否流畅
- [ ] 总时长是否在55-65秒
- [ ] 开头3秒是否有钩子（吸引观众继续看）
- [ ] 结尾是否有悬念（引导看下集）
```

### 6.3 导出
```
点击右上角"导出"
设置：
├── 分辨率：1080×1920
├── 帧率：30fps
├── 格式：MP4
├── 码率：推荐（自动）
└── 导出到：AI短剧/成品/

文件名格式：[故事名]_[日期]_v1.mp4
```

**✅ 验收标准：** 成片导出，1080P竖屏，60秒左右，播放流畅
