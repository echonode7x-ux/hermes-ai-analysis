## 二、赛道选择指南（修正版）

### 2.1 赛道评估矩阵

| 赛道 | 角色一致性 | AI适配度 | 竞争度 | 变现潜力 | 综合推荐 |
|------|-----------|---------|--------|----------|----------|
| 民间传说（奇幻生物） | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 中 | 中 | ⭐⭐⭐⭐⭐ **首选** |
| 科幻机器人 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 低 | 高 | ⭐⭐⭐⭐⭐ **首选** |
| 动物拟人 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 低 | 中 | ⭐⭐⭐⭐ |
| 历史场景（无特定人物） | ⭐⭐⭐ | ⭐⭐⭐⭐ | 中 | 中 | ⭐⭐⭐ |
| 悬疑推理 | ⭐⭐ | ⭐⭐⭐ | 高 | 高 | ⭐⭐⭐ |
| 情感故事 | ⭐ | ⭐⭐ | 高 | 高 | ⭐⭐ |

### 2.2 首选赛道理由

**民间传说（奇幻生物）：**
- 龙/凤/狐仙/妖怪等角色天然不怕"不一致"——本身就是奇幻形象
- AI视频模型对奇幻场景生成效果最好
- 受众广，老少皆宜
- 素材丰富（山海经/聊斋/民间故事）

**科幻机器人：**
- 机器人角色天然"标准化"，一致性问题最小
- AI对科技感/金属质感渲染效果好
- 海外受众接受度高（利于出海）
- 竞争度低

## 三、工具使用细节（修正版）

### 3.1 可灵AI 操作规范

**注册与配置：**
```
1. 访问 klingai.com → 手机号注册
2. 新手任务：完成3个小任务获得约66积分
3. 选择专业版（¥66/月）→ 每日约66积分（约30条5秒视频）
```

**最佳实践：**
```
文生视频：
- 英文提示词效果 > 中文提示词
- 结构：主体+动作+场景+光线+镜头+风格
- 每个镜头生成3条，选最佳
- 5秒/条最稳定，10秒质量会下降

图生视频：
- 先用文生图生成参考图
- 再用"图生视频"让静态图动起来
- 角色一致性比纯文生视频好50%+
- 关键角色必须用图生视频

避坑：
- 避免生成含文字的画面（容易变形）
- 避免手部特写（AI通病）
- 避免复杂多人场景（容易出错）
- 避免快速运动（容易模糊）
```

### 3.2 即梦AI 操作规范

```
1. 访问 jimeng.jianying.com → 抖音账号登录
2. 免费但排队时间长（可能8小时+）
3. 建议：晚上提交任务，第二天早上取结果
4. Seedance 2.0画质最好但算力最紧张
5. 作为可灵AI的备用方案
```

### 3.3 剪映 操作规范

```
核心功能（必须掌握）：
├── 导入素材：视频+图片+音频
├── 时间轴剪辑：裁剪+拼接+分割
├── 文字：添加字幕+标题+花字
├── 音频：AI配音+背景音乐+音效
├── 特效：转场+滤镜+调色
└── 导出：1080P/30fps/竖屏9:16

AI配音选择：
├── 古风/传说：男声-沉稳（语速0.9x）
├── 科幻：男声-科技感（语速1.0x）
├── 悬疑：男声-低沉（语速0.8x）
└── 情感：女声-温柔（语速0.9x）

导出参数：
├── 分辨率：1080×1920（竖屏）
├── 帧率：30fps
├── 格式：MP4
└── 码率：推荐（自动）
```

### 3.4 DeepSeek 剧本生成规范

```
提示词模板：

你是一个60秒短剧编剧。请按以下要求写一个短剧剧本：

题材：[民间传说/科幻/悬疑/情感]
主题：[具体故事主题]

要求：
1. 结构：开头悬念(5秒) → 故事展开(35秒) → 高潮(15秒) → 反转/钩子(5秒)
2. 拆分成6-8个画面，每个画面包含：
   - 视觉描述（用英文，适合AI视频生成）
   - 中文旁白（每句不超过15字）
   - 时长（秒）
3. 旁白节奏：前慢后快，高潮处停顿
4. 避免：复杂多人场景、手部特写、文字画面
5. 角色：使用非人类主角（奇幻生物/机器人/动物）

输出格式：
[画面1] 时长：5秒
视觉描述（英文）：...
旁白（中文）：...
```

## 四、10天MVP执行计划（修正版）

### Phase 1：准备期（Day 1-3）

#### Day 1：注册与工具准备（3小时）
```
上午（1.5h）：
├── 注册可灵AI + 购买专业版（¥66/月）
├── 注册即梦AI（备用）
└── 下载安装剪映

下午（1.5h）：
├── 可灵AI：生成3条测试视频，熟悉操作
├── 剪映：导入测试素材，练习基础剪辑
└── 输出：3条测试视频 + 操作笔记
```

#### Day 2：赛道确定与参考收集（3小时）
```
上午（1.5h）：
├── 在抖音/快手搜索"AI短剧""AI动画"Top20账号
├── 分析：播放量、粉丝数、内容风格、更新频率
└── 输出：竞品分析表

下午（1.5h）：
├── 选定赛道：民间传说 or 科幻机器人
├── 收集10个同赛道优秀作品作为参考
├── 确定视觉风格（写实/动漫/3D）
└── 输出：风格参考库
```

#### Day 3：提示词测试与模板建立（4小时）
```
上午（2h）：
├── 测试10组提示词，找到最佳参数组合
├── 记录：什么提示词出什么效果
├── 建立"提示词模板库"
└── 输出：10组提示词+效果记录

下午（2h）：
├── 用DeepSeek写第一个剧本（测试用）
├── 按剧本生成6个镜头的AI视频
├── 测试"图生视频"流程（先出参考图再出视频）
└── 输出：测试剧本 + 6条视频素材
```

### Phase 2：制作期（Day 4-7）

#### Day 4：正式剧本创作（3小时）
```
全流程：
├── 选定故事（如：《画皮》/《龙女》/《机器人觉醒》）
├── 用DeepSeek生成剧本初稿（15min）
├── 人工调整剧本：确保每个镜头AI可实现（30min）
├── 将视觉描述翻译成英文提示词（30min）
├── 旁白文案定稿（30min）
├── 分镜脚本整理（1h）
└── 输出：完整分镜脚本（含英文提示词+中文旁白+时长）
```

#### Day 5-6：AI视频生成（每天4小时，共8小时）
```
Day 5：
├── 用"图生视频"方式生成关键角色参考图（1h）
├── 逐镜头生成AI视频，每镜头3条选最佳（3h）
├── 产出：8-10条视频素材
└── 注意：错峰生成（早/晚各提交一批）

Day 6：
├── 继续生成剩余镜头（2h）
├── 筛选所有素材，标记"可用/需重做"（1h）
├── 对不合格镜头重新生成（1h）
├── 产出：12-15条最终素材
└── 备用素材归档
```

#### Day 7：剪辑合成（4小时）
```
上午（2h）：
├── 导入所有素材到剪映
├── 按分镜脚本排列时间轴
├── 添加旁白配音（剪映AI配音 or 自己录制）
├── 添加字幕（自动识别+手动校对）

下午（2h）：
├── 添加BGM和音效
├── 添加转场效果
├── 调色统一画面风格
├── 导出1080P竖屏版本
└── 输出：成片1条
```

### Phase 3：发布与优化（Day 8-10）

#### Day 8：发布上线（2小时）
```
├── 注册抖音/快手/视频号创作者账号（如未注册）
├── 上传视频：标题+封面+标签+描述
├── 标注"AI辅助生成"
├── 发布时间：晚7-9点（流量高峰）
├── 评论区引导话术准备
└── 输出：3平台发布完成
```

#### Day 9：数据监测（2小时）
```
├── 检查24小时数据：播放量、完播率、点赞率、评论
├── 记录各平台数据差异
├── 分析：哪个画面观众流失最多
├── 回复评论区互动
└── 输出：首条视频数据报告
```

#### Day 10：复盘与规划（3小时）
```
├── 复盘全流程：耗时、难点、改进点
├── 更新提示词库和模板库
├── 规划第2条视频选题
├── 决定是否继续投入（数据达标则继续，不达标则调整赛道）
└── 输出：10天MVP复盘报告 + 下一步计划
```

## 六、法律红线清单（不变）

```
❌ 绝对不做：
1. AI换脸
2. 使用已有IP（小说/动漫/游戏角色）
3. 生成色情/暴力/政治敏感内容
4. 不标注AI生成直接发布
5. 冒充真人创作

⚠️ 谨慎做：
1. 历史人物题材
2. 仿真人风格
3. 改编已有故事

✅ 安全做：
1. 原创剧本+原创角色
2. 奇幻/科幻/架空世界观
3. 标注"AI辅助生成"
4. 非人类主角（动物/机器人/奇幻生物）
```
