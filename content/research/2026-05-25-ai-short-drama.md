## 五、法律风险深度评估

### 5.1 法规框架

| 法规 | 相关条款 | 对AI短剧的影响 |
|------|----------|----------------|
| 《著作权法》 | AI生成内容的可版权性 | 争议中：纯AI生成可能不受保护 |
| 《生成式AI服务管理暂行办法》 | 标识要求、内容审核 | 必须标注"AI生成"，不得生成违法内容 |
| 《个人信息保护法》 | 肖像权保护 | AI换脸/仿真人需获得授权 |
| 《反不正当竞争法》 | 数据抓取、IP侵权 | 不得使用他人IP进行AI生成 |
| 各平台规则 | 内容标识、质量要求 | 各平台要求不同，需逐一遵守 |

### 5.2 霍去病事件分析

**事件：** AI短剧《霍去病》引发热议，律师指出需警惕风险（潮新闻 2026）

**核心争议：**
1. 历史人物形象的AI再现是否侵权
2. AI生成的历史剧是否歪曲历史
3. 平台是否有权下架

**教训：**
- 避免使用真实历史人物（尤其是近现代）
- 避免使用已有IP（小说/动漫/游戏人物）
- 使用原创角色和剧本

### 5.3 红线清单

```
❌ 绝对不做：
1. AI换脸（将他人面部替换到视频中）— 侵犯肖像权，已有刑事案例
2. 使用已有IP（小说/动漫/游戏角色）— 侵犯著作权
3. 生成色情/暴力/政治敏感内容 — 违法
4. 不标注AI生成直接发布 — 违反《生成式AI管理办法》
5. 冒充真人创作 — 平台违规

⚠️ 谨慎做：
1. 历史人物题材 — 需确保不歪曲历史
2. 仿真人风格 — 需避免与真人过度相似
3. 改编已有故事 — 需确认版权状态

✅ 安全做：
1. 原创剧本+原创角色
2. 奇幻/科幻/架空世界观
3. 标注"AI生成"
4. 教育/知识类AI短剧
```

## 七、提示词库

### 7.1 剧本生成提示词

**通用剧本提示词：**
```
你是一个专业的短剧编剧。请为我写一个1分钟的短剧剧本。

要求：
1. 类型：[历史/科幻/悬疑/情感/民间传说]
2. 结构：开头钩子(3秒) → 冲突建立(20秒) → 高潮(20秒) → 反转/结局(17秒)
3. 画面描述：拆分成5-8个画面，每个画面用详细的视觉描述
4. 旁白/对白：简洁有力，每句不超过15字
5. 情绪曲线：从好奇 → 紧张 → 震撼 → 感悟

输出格式：
[画面1] 视觉描述：...
旁白：...

[画面2] 视觉描述：...
旁白：...
```

**历史题材提示词：**
```
请以[朝代]为背景，写一个关于[人物/事件]的1分钟短剧剧本。

要求：
- 还原历史氛围（服饰、建筑、场景）
- 人物有血有肉，不是脸谱化
- 有一个出人意料的反转
- 画面描述要适合AI视频生成（具体、可视化）
```
