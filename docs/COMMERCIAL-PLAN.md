# HERMES AI Analysis 商业化版图 v1.0

## 一、定位升级

**从"研究站"升级为"AI一人公司商业情报平台"**

- 不只是展示信息，而是帮用户**做决策、拿结果**
- 每条内容 = 一个可执行的赚钱方案
- 用户路径：看情报 → 做决策 → 拿结果 → 付费进阶

## 二、内容管线（每日自动产出）

### 2.1 每日信号采集（凌晨2点，自动）
- 扫描 GitHub Trending / Product Hunt / 即刻 / V2EX
- 筛选标准：成本<500、单人可做、有真实案例
- 输出：信号池 + 公众号日更文章

### 2.2 每日公众号文章（上午10点，自动推送）
- 从信号池中选1个最有价值的方向
- 输出1500-2000字的公众号文章
- 标题公式：[具体结果] + [时间/成本] + [钩子]
- 内容：机会拆解 + 技术栈 + 成本 + 执行步骤 + 风险

### 2.3 每周深度研究（周三/周日，自动推送）
- 从信号池中选1个方向，输出完整可执行方案
- 包含：市场验证、技术栈、成本清单、7天MVP、收益模型、风险
- 保存到项目的 research 目录

### 2.4 每月复盘报告（每月1号，自动推送）
- 回顾本月输出的方案
- 哪些被验证有效，哪些被淘汰
- 下月方向调整建议

## 三、站点内容架构（从6页扩展到20+页）

### 3.1 核心页面（已有）
- 首页 / 机会库 / 赛道 / 关键词 / 信号 / 后做 / 方法

### 3.2 新增页面（商业化必需）
- **每日情报**：展示每日采集的信号，形成信息流
- **深度研究**：展示完整的研究方案，可按赛道筛选
- **工具箱**：推荐AI工具，带联盟分销链接
- **模板库**：Prompt模板、Rules模板、工作流模板
- **案例库**：真实用户案例，带收入数据
- **会员专区**：付费内容入口
- **留资页**：企业咨询、选型服务

### 3.3 内容分类
| 类型 | 频率 | 目的 |
|------|------|------|
| 每日信号 | 每天 | SEO + 流量 |

| 工具评测 | 每周1篇 | 联盟收入 |
